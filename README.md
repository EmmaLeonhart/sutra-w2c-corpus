# sutra-w2c-corpus

Weights<->code training corpus for **Sutra** weight->code decompilation.

Each `corpus.jsonl` line is one datapoint: a generated trainable-matrix Sutra program (`source`), its file-backed weight matrices (`weights[]` -> CSV files here), and the input->output `io[]` the program produces **on the Sutra substrate**. Generated by `experiments/weight_to_code_corpus.py` in the Sutra repo (this is pinned there as a submodule). Mirrored to Hugging Face.

Program structures (v0): linear, chain2, chain3, residual, diff, scaled, affine, sum2, bundle2, bundle3. Weights: randomized (gaussian/perm); trained variants forthcoming. Model-free (no embedding LLM); each entry is self-consistency-tested (recompile source+weights, reproduce io on the substrate).
